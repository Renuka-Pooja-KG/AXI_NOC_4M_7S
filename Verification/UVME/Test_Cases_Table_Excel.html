Status Legend

  • Not Started: Test case has not been executed yet
  • In Progress: Test case is currently being executed
  • Completed: Test case has been executed successfully
  • Failed: Test case failed during execution
  • Blocked: Test case is blocked due to dependencies

Priority Legend

  • High: Critical functionality, must pass for release
  • Medium: Important functionality, should pass for release
  • Low: Nice-to-have functionality, can be deferred

Usage Instructions

  1. Select the entire table above (Ctrl+A)
  2. Copy the selection (Ctrl+C)
  3. Open Excel and create a new workbook
  4. Paste the table (Ctrl+V)
  5. The table will be automatically formatted in Excel
  6. Use the Status column to track progress
  7. Add dates and tester information as tests are executed
